# Get started with Codebuff

Follow these steps to install and run Codebuff in your project.

## 1. Install Codebuff

Enter the following in your terminal, which could be inside your favorite IDE (VSCode, Cursor, etc.).

```bash
npm install -g codebuff
```

## 2. Navigate to your project directory

```bash
cd /path/to/your-repo
```

## 3. Run Codebuff

```bash
codebuff
```

Codebuff has multiple [modes](/docs/tips#modes): `--lite` for quick tasks, `--max` for complex work, and `/mode:plan` for planning without file changes.
